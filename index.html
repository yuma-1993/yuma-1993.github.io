La importancia de un perfil interdisciplinar

Estas son las disciplinas en las que me he estado especializado. Considero que tener un perfil interdisciplinar significa conseguir una ventaja competitiva, aportando valor al equipo de trabajo

Sociología

Está disciplina tiene como objeto de estudio el comportamiento humano, así como aquellos grupos sociales e instituciones que componen la sociedad. Utiliza metodologías de producción de datos y análisis tanto cuantitativas como cualitativos.

Análisis de datos

Es una ciencia encargada de examinar los datos, transformarlos y convertirlos en información para se utilizados para la toma de decisiones. Se emplean metodologías tanto cuantitativas como cualitativas, para la extracción procesamiento y análisis.

Programación

La tecnología avanza, y así también lo hacen las herramientas para realizar las infraestructuras para el almacenamiento y trasmisión de datos, así como para el análisis. En este sentido surgen diversos lenguajes de programación que son óptimos para la ciencia de datos.
